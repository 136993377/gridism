gridism.

A simple, responsive, and handy CSS grid.

View on GitHub Download

Simple

Each.grid element is a horizontal row which can contain .unit elements of various widths. Width combinations are illustrated below.

Responsive

Units get stacked on screens 568px or smaller. Grids in a .wrap have maximum widths of 978px, or 1140px when there’s lots of room.

Handy

When units are given .pad-h (horizontal) and .pad-v (vertical) classes, content gets even padding and gutters. There are also a range of text-alignment classes.

Installation

There are three ways you can have gridism be part of your project.

  • Install using Bower:
    bower install gridism
  • Download the zipball from GitHub
  • Clone the repository
