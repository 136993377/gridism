gridism.

A simple, responsive, and handy CSS grid by @cobyism.

View on GitHub Download

Simple

Each.grid element is a horizontal row which can contain .unit elements of various widths. Width combinations are illustrated below.

Responsive

Units get stacked on screens 568px or smaller. Grids in a .wrap have maximum widths of 978px, or 1140px when there’s lots of room.

Handy

All units have even inner and outer gutters. You can turn gutters off with .no-gutters. There are also a few of other commonly needed classes.

Installation

There are three ways you can have gridism be part of your project.

Bower

Gridism can be installed using the Bower package manager.

bower install gridism

Download

Get the source code, extract it, and copy css/gridism.css into your project’s directory.

GitHub

Clone the repository using your terminal, or even add it as a submodule of your project.

Linking the stylesheet

Once you’ve acquired the gridism source code, add the following stylesheet to the <head> section of your project’s HTML.

<link rel="stylesheet" href="components/gridism/gridism.css">

If you didn’t use Bower, you may need to adjust the path to the CSS file to match your file structure
